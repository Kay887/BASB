🍎Apple Calendar (CalDAV) Not connected
Generate an app-specific password at appleid.apple.com → Sign-In and Security → App-Specific Passwords, then paste it here with your Apple ID email. Kept in this browser tab's session only — never synced to any database.
⚠ Honest limit: Apple's CalDAV server blocks direct browser requests (no CORS support), and there's no public proxy for the CalDAV protocol the way there is for RSS. Your credential is saved here, but live event fetching from this page isn't possible without a backend server — that's a real wall in how browsers and Apple's servers talk to each other, not a missing feature.
